在评论区使用markdown语法

2019-03-22 发布在 web语言

markdown是一款轻量级网页内容生成语法，使用markdown语法，你可以轻松生成图文并茂的网页内容。这篇文章将带你了解如何在评论区使用markdown语法。

1、markdown语法

不同的markdown语法解释器对于markdown语法的支持会略有不同，但基本语法都大体相似。这里我们使用的gitment评论系统使用的是github的markdown语法解释器，你可以在 这里查看github对其markdown语法解释器的官方说明。

以下内容是根据我个人的理解对markdown语法的归纳总结。其中，在代码与效果对比框中，背景色为灰色的部分是markdown语法格式，背景色为白色的部分是markdown语法效果。

1.1、标题语法

你可以使用# 标题名称的格式插入标题，其中#的数量代表标题级别，这里仅支持一至六级标题。

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

1.2、文字、段落语法

你可以使用**粗体字**或者__粗体字__的格式加粗字体，使用*斜体字*或者_斜体字_的格式使字体倾斜，使用~~删除线~~的格式在文字中插入一条删除线，使用`行内代码` 的格式插入行内代码，使用[超链接](链接地址)的格式插入超链接。

对于段落而言，如需新开一行必须在上一行末尾插入两个空格，如需新开一个段落则需要先插入一个空行，注意markdown不支持通过换行符插入多个空行。

- **粗体字**, __粗体字__
- *斜体字*, _斜体字_
- ~~删除线~~
- `行内代码`
- [超链接](https://wly.supernum.tech)
  • 粗体字, 粗体字
  • 斜体字, 斜体字
  • 删除线
  • 行内代码
  • 超链接

1.3、引用语法

你可以使用> 应用内容的格式插入引用的内容，引用的时候请注明来源。

> 引用他人的言论或网页、书籍中的内容
> ——引用来源

引用他人的言论或网页、书籍中的内容 ——引用来源

1.4、代码块语法

如果你需要插入多行代码，可以使用```作为代码的首行和尾行，如需高亮代码，则可以在首行的```之后标明代码类型。

```html
<html>
  <head>
    <meta charset="UTF-8">
    <title>Document</title>
  </head>
  <body>
    Hello World！
  </body>
</html>
```

1.5、图片语法

你可以使用![图片Alt](图片链接地址)的格式插入一幅图片，注意图片与超链接的语法区别。你也可以直接拖放一张图片至评论区，图片代码将会自动生成。

![图片Alt](https://s2.ax1x.com/2019/08/04/eyNtUI.png)

1.6、列表语法

你可以使用数字编号. 列表内容的格式插入有序列表，使用- 列表内容的格式插入无序列表，使用- [ ] 列表内容的格式插入任务列表，如需表示任务已完成，可以在方括弧内输入一个英文字母x。如需插入多级列表，则需要在子列表之前输入两个空格。

  • 有序列表
1. 列表内容1
2. 列表内容2
3. 列表内容3
  1. 列表内容1
  2. 列表内容2
  3. 列表内容3
  • 无序列表
- 列表内容1
  - 子列表内容1
  - 子列表内容2
  • 列表内容1
    • 子列表内容1
    • 子列表内容2
  • 任务列表（gihub特有）
- [x] 已完成任务
- [ ] 未完成任务
  • 已完成任务
  • 未完成任务

1.7、表格语法

如需插入表格，则需要以 | 来区分不同的列，以换行来区分不同的行，以多个-来区分表头与表格内容，以:--、--:、:--:来分别表示左对齐、右对齐和剧中对齐。

|             | 占位符       |

1.8、其它github特有的markdown语法

1.8.1、回应某人

你可以使用@github账户名的格式，来特指某人。这种方式本质上只是生成了一个指向某个github用户主页的超链接，所以你也可以直接使用超链接的格式达到相同的效果。

1.8.2、使用Emojj表情

你可以使用:Emojj名称:的格式插入Emojj表情，比如使用see_no_evil、 hear_no_evil、 speak_no_evil三个Emojj名称，分别插入非礼勿视🙈、非礼勿听🙉、非礼勿言🙊。你可以在 这里查看评论区支持的所有Emojj名称。

2、使用html标签

markdown本质上只是将你输入的内容转化为html语言，如果你觉得现有的markdown语法不够丰富，则完全可以直接通过html标签来实现更多功能。

2H<sub>2</sub> + O<sub>2</sub> = 2H<sub>2</sub>O
2H2 + O2 = 2H2O
